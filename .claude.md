# Windows 避坑指南

AI 驱动的 Windows 开发/运维踩坑知识库。

## 使用流程

当用户遇到 Windows 相关问题（编码、注册表、MSYS、提权等）：

1. **先查** — 运行 `bash ~/.claude/scripts/pitfalls.sh search '<JSON>'` 搜索已知坑
2. **如果命中** — 直接给出解决方案
3. **如果没命中** — 比对本地已有条目（去重检测）
4. **确实没有** — git fetch 检查远程，有变化才 pull + 重建索引 + 重搜
5. **还是没有** — 运行 `bash ~/.claude/scripts/pitfalls.sh record '<JSON>'` 记录新坑
   - 脚本自动完成：写 .md → 重建索引 → git commit → git push (add/ 分支) → gh PR → GitHub Actions 自动合入
   - 整个过程约 10-15 秒

注意：只在问题**解决后**一次性记录（症状+根因+方案全），不记"待解决"坑。安装步骤见 [README](README.md)。

### 搜索 (pitfalls.sh)
```bash
bash ~/.claude/scripts/pitfalls.sh search '{"query":"关键词","severity":"medium","limit":5}'
```
参数: query(必填), tool, os, severity(enum:critical/medium/low), limit(默认10)

### 记录 (pitfalls.sh)
```bash
bash ~/.claude/scripts/pitfalls.sh record '{"title":"坑标题","symptom":"现象","root_cause":"根因","solution":"方案","tool":"相关工具","os":"Windows 10/11","severity":"medium","tags":"tag1, tag2"}'
```
参数: title/symptom/root_cause/solution(必填), tool/os/severity/tags(可选)

项目结构见 [docs/plan.md](docs/plan.md)。
