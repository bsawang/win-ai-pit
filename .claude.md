# Windows 避坑指南

本项目是一个 **AI 驱动的 Windows 开发/运维踩坑知识库**。基于 pyrite 框架，通过 MCP
协议为 AI 提供结构化的坑检索和记录能力。

## 使用流程

当用户遇到 Windows 相关问题（编码、注册表、MSYS、提权等）：

1. **先查** — 调用 `search_pitfall` 工具搜索已知坑
2. **如果命中** — 直接给出解决方案
3. **如果没有命中 / 用户纠正** — 调用 `record_pitfall` 工具记录新坑（自动去重）

## 启动 MCP Server

```bash
cd E:\work\ai\win-ai-pit

# 1. 重建索引（新增/修改文件后需要）
python -c "from pyrite.config import load_config; from pyrite.storage.database import PyriteDB; from pyrite.storage.index import IndexManager; c=load_config(); d=PyriteDB(c.settings.index_path); i=IndexManager(d); [i.index_kb(kb.name) for kb in c.knowledge_bases]; d.close()"

# 2. 启动 MCP Server
python -m pyrite.server.mcp_server --tier admin
```

## 项目结构

```
├── pyrite/                     ← pyrite 核心源码（MIT，保留版权）
├── pyrite_windows_pitfalls/    ← 我们的插件（原创）
│   └── plugin.py               ← MCP 工具（search_pitfall, record_pitfall）
├── pitfalls/                   ← 坑内容（markdown）
│   └── msys/nul-to-dev-null.md ← 示例坑
├── kb.yaml                     ← pyrite 配置（windows_pitfall type）
├── data/index                  ← SQLite 索引（自动生成）
└── pyproject.toml
```

## MCP 工具

### search_pitfall
搜索已知坑。支持按 query、tool、os、severity、limit 过滤。

### record_pitfall
记录新坑。自动去重（按标题/症状相似度匹配）。参数：title, symptom, root_cause,
solution, os, tool, tool_versions, severity, tags。
